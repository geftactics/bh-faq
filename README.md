## Installation

- Install [Ruby](https://www.ruby-lang.org/en/downloads/) via download or Homebrew `brew install ruby`
- Install [Jekyll](https://jekyllrb.com/docs/installation/macos/)
- cd to this repository
- Run `bundle install`
- Run `bundle exec jekyll serve`
